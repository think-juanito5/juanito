# @dbc-tech/asreplica

Drizzle schema & queries to connect to Actionstep replica database

# Setup

Uses [bun](https://bun.sh/) instead of Node

Install Bun

```bash
curl -fsSL https://bun.sh/install | bash
```

Install Packages

```bash
bun install
```

Build, test, lint

```bash
bun check:all
```
